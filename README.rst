aliyunhpaplugin plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://codeup.aliyun.com/fira/fira/tutor-contrib-aliyunhpaplugin

Usage
-----

::

    tutor plugins enable aliyunhpaplugin
